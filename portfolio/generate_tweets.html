GPT-2 Tweet Generator

Train GPT-2 to generate tweets based on the tweets from #66DayofData-challenge

Programming language: Python
Libraries: GPT-2-simple, tensorflow, spaCy, google's universal sentence encoder, pandas

Brief Overview:

Goal

  • Training GPT-2 model and using it to generate tweets

Methodology

  • Process data to make it suitable for the model
  • Trained 124M base model
  • Checked generated tweets for similarity

Results

  • Trained GPT-2 on almost 15000 tweets (2000 epochs)
  • Generated tweets were unique (model didn’t memorize input tweets)

What did I learn?

  • Working with a pre-trained ml-model
  • Testing similarity between generated and genuine tweets using spaCy similarity score and google’s universal sentence encoder

ToDo

  • Retrain with newly collected tweets
  • Use different GPT-2 model (355M instead of 124M base model)
  • Fine tune parameters for generating tweets
  • Make an application where users can create tweets with a click

What is GPT-2 ?

GPT-2 is a large transformer-based language model with 1.5 billion parameters, trained on a dataset of 8 million web pages (40GB text data).
